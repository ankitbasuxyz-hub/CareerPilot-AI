# 🌟 Overview

CareerPilot AI is an intelligent **Multi-Agent AI Career Assistant** designed to help students and professionals prepare for their dream careers.

Instead of relying on multiple websites, CareerPilot AI combines **Resume Analysis, Skill Gap Detection, Personalized Learning Roadmaps, Mock Interviews, and Career Guidance** into one unified platform.

The application runs locally, providing a fast, privacy-first experience.

---

# ✨ Features

| Feature | Description |
|----------|-------------|
| 📄 Resume Review | Resume readiness score, missing skills, ATS improvements |
| 🧠 Learning Roadmap | Personalized learning plans and milestones |
| 🎤 Mock Interview | AI-generated interview practice with feedback |
| 💼 Career Advisor | Career guidance, salary insights & planning |
| 📊 Dashboard | Resume score, skill gap analysis & progress tracking |
